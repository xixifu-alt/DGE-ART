基于方向引导探索的自适应随机测试（Direction-Guided Exploration ART, DGE-ART）

自适应随机测试（ART）通过增强测试用例分布的均匀性来提升故障检测能力。然而，传统的 ART 算法在程序维度增加时，其计算复杂度呈指数级增长，从而限制了其实用性。
为解决这一问题，本文提出了一种新方法：基于方向引导探索的自适应随机测试（Direction-Guided Exploration ART, DGE-ART）。该方法在初始故障检测位置附近探索第二个故障点，并以此建立方向引导，用于指导对下一个故障区域的搜索。该方法特别适用于数值程序中常见的条带状故障模式。

主要优势：

降低计算开销：在低维空间中显著减少计算成本。

 高维高效：随着维度增加，故障检测效率进一步提升。
 
鲁棒性强：在不同故障率下均表现稳定，验证了其广泛的适用性。

意义：

本研究不仅为解决 ART 在高维场景下面临的“维度灾难”问题提供了创新思路，也为提升软件测试的智能化和自适应水平开辟了新的方向。
